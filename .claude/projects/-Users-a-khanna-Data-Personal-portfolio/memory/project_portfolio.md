Personal portfolio site built with Next.js (App Router), TypeScript, Tailwind CSS, and Three.js (FloatingScene).

Key sections: Hero, About, WhatIDo, Career, Projects, TechStack, Contact.
Has admin panel at `/admin-panel-9x7k/` with auth API routes.
Data driven from `src/data/portfolio.json`.
Resume page with print support.
Custom UI: cursor, scroll reveal, glow effects, theme toggle (dark/light).
